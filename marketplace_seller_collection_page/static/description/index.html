Odoo Marketplace Seller Collection Page

Odoo Marketplace Seller Collection Page is a module by which sellers of marketplace can make collections of products and publish it on website. Sellers can create and edit their own collection pages.

Highlighted Features

  • Admin and Seller both can create collection pages for more sales.
  • Admin can add any product of marketplace in the collections.
  • Sellers can add only their own products in the collections.
  • Admin and Seller can Publish/Unpublish collection pages and display them on the website according to requirement.
  • SEO Management for each collection page.
  • Admin and Seller both can make collection groups by adding collection pages into it.
  • Collection groups can be displayed in website in Single Carousel or Multi Carousel form.
  • Admin can manage seller-wise whether to allow seller to make collections or not.
  • All the collections of a particular seller can be seen under Collection Tab in Seller Profile in website.
