Odoo Marketplace Advance Commission

Odoo Marketplace Advance Commission allows admin to set commission on seller products, product categories and sellers of marketplace.

Highlighted Features

  • Admin can set commission on products.
  • Admin can set commission on product categories.
  • Admin can set commission on seller profile also.
  • Admin can decide different commission like percentage or fixed for products, categories and seller.
  • Commission on product will get higher priority.
  • Commission on product is not set then commission on the product category will get priority.
  • Commission on product category is not set then commission on the seller profile will be applied.
  • Admin can decide whether to deduct the minimum or maximum commission rate, if a product has more than one category.
  • Decide which of the commission type i.e. category, product or default commission will be applied on orders.
  • Admin can apply the commission criteria commonly to all sellers.
  • Admin can also set different types of commission for different sellers of marketplace.
  • Automatic deduction of commission from the seller payment.
  • Seller will receive the payment after deducting the commission.

Note:

  • Admin can set the category commission to be applied if product is under more than one category.
  • Admin can set the default commission method commonly for all sellers from Configuration settings.
  • Admin can set the global, fixed, global + fixed, fixed + global commission in percentage for all sellers.
