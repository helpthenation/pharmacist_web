ODOO Warehouse Order Split Management
For Providing Easy Way To Managing Multiple Shipment

Few Words About Order Split:

Order Split Plug-in Provide an easy interface for managing the move line in side warehouse.

Some time due to unavailability of products ,providing the delivery of order on the committed time become a tedious issue in such case the warehouse manager can use this pulg-in to make the order delivery in multiple batches.

When the customer order's more then the availability of the product in warehouse the seller can split the order and deliver available products at that time and rest of them in latter with a new warehouse reference.

Work Flow

Sale Order Creation From Website

Steps For Order Splitting

Select the order you want to split and press (split)

After pressing the Split button the wizard will open .

Edit the quantity or Delete the Move Line as per requirements and press(split) button

There can be many cases

Example(s):

  1. If you want to delivered 5 iPad Retina Out of 10 latter then just edit it from 10 to 5 .
  2. If you want to delivered Only iPad Mini latter then delete other except iPad Mini.
  3. If you want to delivered All 5 IMac At Once latter then delete other except IMac.

Splitting Case 1

Deliver 5 iPad Retina Latter

Split-ed Delivery View

Newly Create First Delivery Order

After splitting now you can deliver 5 iPad Retina latter with a new warehouse reference[WH/OUT/00010] having [WH/OUT/00008] as back order reference.

Newly Create Second Delivery Order

Rest of these product are now ready to deliver with [WH/OUT/00009] warehouse reference having [WH/OUT/00008] as back order reference.
