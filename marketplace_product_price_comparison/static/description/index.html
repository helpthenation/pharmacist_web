Odoo Marketplace Seller Price Comparison
Manage product multi seller features for your marketplace.

Note: This module is an addon for Odoo Multi Vendor Marketplace module. The price reflected here includes addon cost plus all dependent modules cost.

Features:

  • Admin can create global products
  • Seller can link their product to global product
  • Sellers list selling same global product will be displayed on product page
  • Admin can manage view of products on shop page as per global/lowest-highest price or can set manually
  • Seller can specify the price and quantity of the product linked with global products
  • Display different seller in product page with their price
  • Buyer can compare same product of different sellers

Odoo Marketplace Seller Price Comparison Use Case -

  • After the installation of Odoo marketplace module, the marketplace seller comparison module can be installed, as this module is an add on for the Odoo marketplace module
  • Now as the marketplace seller comparison module is installed, the admin can set any of the product as global from the backend
  • Once the admin has set the product as global, then various sellers can assign product to the global products set by the admin
  • Since different sellers have assigned many products to the global product set by the admin. So when the buyer will look for the product then at the product detail page, can view all the other products by various sellers

    • This way buyer can make the comparison between different seller products
