Salient Features

  • Send an e-mail notification for recovery of abandoned cart.
  • Recover the Lost Sales with Abandoned cart recovery module.

  • Cron settings are available to automatically send an email to customer’s abandoned checkout.

Abandoned cart recovery configurations

Under settings, you will find an option to do the configurations for abandoned cart recovery.

Cron settings

On the abandoned cart recovery configurations page, you will find an option for cron settings. Do the configuration settings, set the number of intervals with units, to define time duration in which cron scheduler will run automatically.

Abandoned order detailed view

A separate list of abandoned orders is maintained where admin can check individually, which cart is abandoned by which customer having what products.

Abandoned cart view

As a registered customer, do add to cart and give up along the way and doesn’t complete with the checkout process, in that case cart is considered to be an abandoned cart after certain duration of time declared from the back panel.

Customers page showing abandoned cart information

As you view the customer page, there you will find an option of cart information, under which the current order number is displayed showing products added to the cart.
