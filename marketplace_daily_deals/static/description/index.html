Odoo Marketplace Daily Deals

Basic Features

  • Marketplace Daily Deals is used to create offers/deals in the Marketplace.
  • Sellers in a Marketplace can create their own deal.
  • Sellers can add only their own products in the deal.
  • Admin can create a deal of any of the seller's product related to the marketplace.
  • Seller can set pricelist when creating new Daily deal/ offer.
  • Seller can also Customize website view as per his need.
  • Seller can manage Message and Notification settings.
  • It gives a separate menu for creating only deals and offers.
  • Only validated deals are displayed on the website.
  • A deal will have a start date and expiry date.
  • Displayed on the website in a separate menu 'Deals and Offers'.

Marketplace Daily Deals

Seller Side View

Deals and Offers menu gets visible to seller .

Seller can create the deal.

Seller can enter the details of the deal/offer.

Seller can set pricelist, manage message and notification settings.

Seller can also customize website view.

Seller has to validate the deal to display it on website.

Status of deal changes to 'In Progress'

Seller can add only his own approved products in the deal.

Products added in the deal.

Seller can view all the deals created by him only.

Admin Side View

A new Deals and Offers menu is visible in a Seller Dashboard.

Admin can create the deal.

Admin can enter the details of the deal.

Products to the deals can be added .

Admin can add any product of any seller of marketplace in the deal.

Admin can view the deals of all the sellers in the marketplace.
